# 星立 一个Vue的UI组件库

作者： Xin-hai

![Static Badge](https://img.shields.io/badge/bulid-passing-brightgreen)

## 介绍
这是我在学习Vue之后，实现的一款基于Vue技术栈实现的极简UI组件库。

## 开始使用

1. css样式

    使用本框架，默认CSS使用border-box

    ```css
    *,*::before,*::after{box-sizing:border-box}
    ```
2. 安装xing-li
    ```
   npm i --save xing-li
    ```
3. 引入xing-li
4. 开发中
